# Périmètre MVP réaliste

Le MVP doit couvrir proprement les domaines suivants.

## Viewer MVP

- auth,
- homepage,
- watch page,
- search,
- channel page,
- subscribe,
- comments,
- likes,
- playlists,
- watch later,
- history.

## Creator MVP

- creator onboarding,
- upload vidéo,
- traitement statut vidéo,
- gestion bibliothèque,
- édition titre / description / tags / miniature,
- privacy,
- analytics simples.

## Admin MVP

- modération vidéos,
- modération commentaires,
- gestion signalements,
- tableau de bord simple.

## Infra MVP

- stockage objets,
- transcoding async,
- thumbnails,
- streaming HLS,
- CDN,
- DB relationnelle pour métadonnées,
- cache,
- queue jobs,
- observabilité.

## Hors MVP mais architecture prévue pour

- live streaming,
- shorts,
- ads,
- monetization,
- memberships,
- copyright automation avancée,
- recommandations ML poussées,
- multi-audio / sous-titres avancés.
