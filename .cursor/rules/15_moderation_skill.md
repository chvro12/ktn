# Moderation & safety

Toute vraie plateforme vidéo doit prévoir :

- signalement vidéo,
- signalement commentaire,
- review queue,
- actions admin,
- historique d’actions,
- raisons standardisées,
- suppression / blocage / restoration.

## États possibles d’un report

- OPEN
- IN_REVIEW
- RESOLVED
- REJECTED

## Actions modération vidéo

- remove visibility,
- block playback,
- warning,
- strike,
- delete soft,
- delete hard admin only.

Toujours garder auditabilité.
