# Prompt spécial refactor

Quand tu refactors le code :

- n’effectue pas de refactor cosmétique inutile,
- ne casse pas le comportement métier,
- améliore lisibilité, séparation des responsabilités et typage,
- réduis duplication,
- garde les APIs stables si possible,
- explique les changements structurels,
- signale les risques de régression,
- fournis une migration claire si nécessaire.
