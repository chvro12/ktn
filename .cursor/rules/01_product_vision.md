# Vision produit — Plateforme vidéo

Construire une plateforme vidéo avec les modules suivants.

## Côté public / viewer

- page d’accueil avec feed personnalisé ou éditorial,
- page watch vidéo,
- recherche,
- pages catégorie / sujet,
- page chaîne / créateur,
- commentaires,
- abonnements,
- playlists,
- historique,
- watch later,
- likes,
- recommandations,
- mini-player,
- file d’attente,
- notifications.

## Côté créateur

- upload vidéo,
- édition metadata,
- miniature,
- visibilité (public / privé / non listé),
- gestion des vidéos,
- analytics créateur,
- gestion des commentaires,
- playlists,
- branding chaîne,
- organisation contenu.

## Côté modération / trust & safety

- signalements,
- review queue,
- blocage / retrait,
- shadow actions éventuelles,
- gestion copyright simple,
- audit log.

## Côté admin

- gestion utilisateurs,
- gestion créateurs,
- gestion vidéos,
- modération,
- analytics globaux,
- catégories,
- tags,
- pages CMS simples.

Le produit doit être pensé comme un vrai service vidéo : ingestion vidéo, transcodage, thumbnails, HLS/DASH, CDN, suivi de vues, ranking, recherche, sécurité.
