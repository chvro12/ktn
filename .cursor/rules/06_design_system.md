# UX / UI system

Le design doit évoquer une vraie plateforme média premium.

## Principes visuels

- fond neutre, contrasté mais pas agressif,
- cartes vidéo sobres,
- layout dense mais respirant,
- très bonne hiérarchie du texte,
- miniatures mises en valeur,
- sidebar propre,
- topbar réaliste,
- onglets, filtres, chips, menus cohérents,
- responsive sérieux.

## Références de qualité

- YouTube pour la logique produit,
- Linear / Raycast / Vercel pour la netteté,
- Apple / Notion / Attio pour la retenue,
- Netflix / Vimeo pour la mise en scène média.

## À éviter absolument

- trop de dégradés,
- trop de glassmorphism,
- cards gonflées inutilement,
- boutons énormes,
- trop d’animations,
- effets gadgets.

## Composants à prévoir

- app shell,
- sidebar desktop / mobile,
- top navigation,
- search bar avancée,
- video card variants,
- channel row,
- comment item,
- playlist item,
- chips de filtre,
- dropdown menus,
- context menus,
- modals et sheets,
- skeleton loaders,
- empty states,
- error states,
- player controls,
- analytics cards crédibles.

## Typographie

- titres nets et compacts,
- metadata secondaires discrètes,
- densité similaire à un vrai produit web,
- line-clamp propre.
