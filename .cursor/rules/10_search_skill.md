# Search

La recherche doit gérer :

- vidéos,
- chaînes,
- playlists,
- tri,
- filtres,
- pagination,
- tolérance de frappe.

## Filtres MVP

- type de contenu,
- date upload,
- durée,
- catégorie,
- ordre (relevance / newest / views).

## Search UX

- suggestions dans la barre,
- historique récent,
- résultats structurés,
- cards adaptées selon type.
