# Prompt spécial UI

Quand tu conçois l’UI :

- pense produit réel, pas concept shot,
- évite les décorations gratuites,
- privilégie les layouts utilisés par de vraies plateformes média,
- travaille la hiérarchie des miniatures, titres, metadata et actions,
- fais des composants crédibles en usage quotidien,
- garde une cohérence entre homepage, watch page, chaîne et studio,
- les interfaces admin et studio doivent être sobres et professionnelles,
- les cartes vidéo doivent être premium mais discrètes,
- les tailles, espacements et alignements doivent être impeccables.

Références d’esprit : YouTube, Vimeo, Netflix web, Linear, Vercel, Attio. Pas de rendu Dribbble gratuit.
