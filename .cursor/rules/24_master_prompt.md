# Prompt d’implémentation global pour Cursor

Construis une plateforme vidéo web moderne inspirée des vrais principes d’un site type YouTube.

## Contraintes absolues

- rendu premium, moderne, crédible, pas « AI generated », pas « vibe coded »,
- architecture réaliste séparée par domaines métier,
- code TypeScript propre et maintenable,
- UI dense, nette, produit réel,
- flows complets viewer + creator + admin,
- state models explicites,
- gestion des loading/error/empty states,
- responsive sérieux,
- accessibilité minimale solide,
- SEO sur pages publiques,
- backend pensé pour upload, processing, playback, engagement, search et modération.

## Ordre de travail

1. proposer l’architecture globale,
2. proposer le schéma de données,
3. proposer la structure du projet,
4. générer le design system de base,
5. construire les pages viewer critiques,
6. ensuite le studio créateur,
7. ensuite l’admin/modération.

Pour chaque étape :

- explique brièvement le rôle métier,
- code proprement,
- n’invente pas de faux patterns visuels,
- garde un niveau production-ready.
